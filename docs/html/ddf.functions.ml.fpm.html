ML.Frequent Pattern Mining¶

class ddf.functions.ml.fpm.Apriori(column, min_support=0.5)¶

Bases: object

Apriori is a algorithm to find frequent item sets which carries out a breadth first search on the subset and determines the support of item sets by subset tests.

Example: 
>>> apriori = Apriori(column='col_0', min_support=0.10).run(ddf1)
>>> itemset = apriori.get_frequent_itemsets()
>>> rules = apriori.generate_association_rules(confidence=0.1)

Setup all Apriori’s parameters.

Parameters:
  • column – Transactions feature name;
  • min_support – minimum support value.

class ddf.functions.ml.fpm.AssociationRules(col_item='items', col_freq='support', confidence=0.5, max_rules=-1)¶

Bases: object

Association rule learning is a rule-based machine learning method for discovering interesting relations between variables in large databases. It is intended to identify strong rules discovered in databases using some measures of interestingness.

Example: 
>>> rules = AssociationRules(confidence=0.10).run(itemset)

Setup all AssociationsRules’s parameters.

Parameters:
  • col_item – Column with the frequent item set (default, ‘items’);
  • col_freq – Column with its support (default, ‘support’);
  • confidence – Minimum confidence (default is 0.5);
  • max_rules – Maximum number of output rules, -1 to all (default).
